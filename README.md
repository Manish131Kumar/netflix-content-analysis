# Netflix Content Strategy Analysis

## Project Overview

This project analyzes Netflix's Movies and TV Shows dataset to uncover patterns and generate business insights that can help Netflix decide what type of content to produce and how to expand across different countries.

## Business Problem

Netflix wants to understand audience trends and identify content opportunities across regions. The goal is to use data analysis to support strategic content decisions.

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Analysis Performed

* Data Cleaning and Preprocessing
* Missing Value Detection
* Statistical Summary
* Univariate Analysis
* Bivariate Analysis
* Country-wise Content Analysis
* Genre Distribution Analysis
* Outlier Detection
* Visualization and Insights

## Key Insights

* Movies dominate the Netflix catalog compared to TV shows.
* Certain countries contribute significantly more content than others.
* Content trends vary across regions and genres.
* Netflix can focus on region-specific content strategies for business growth.

## Business Recommendations

* Invest in high-demand genres.
* Expand localized content production.
* Target emerging markets with region-specific content.
* Increase focus on successful content categories.
